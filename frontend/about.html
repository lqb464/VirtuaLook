Về VirtuaLook

VirtuaLook là engine thử đồ ảo (Virtual Try-On) dùng diffusion inpainting. Dự án tập trung vào ML inference service (services/vton/app) với giao diện demo HTML/JS mỏng để exercise API.

Pipeline

CatVTON: ghép ảnh người + trang phục cạnh nhau → SD v1.5 inpainting UNet (attention weights fine-tuned) → crop vùng người → kết quả cuối.

Roadmap

  • DensePose / SCHP masks (weights đã có trên HuggingFace, chưa tích hợp)
  • Upload ảnh trang phục trực tiếp (hiện chỉ URL)
  • Job queue với worker pool
  • Local IDM-VTON weights (stub hiện tại)

Tech stack

FastAPI · PyTorch · Diffusers · SQLAlchemy · SQLite · Replicate API · Vanilla HTML/JS
