Hướng dẫn cài đặt & sử dụng

1. Cài đặt nhanh (mock — không cần GPU)

cd VirtuaLook
cp services/vton/.env.example services/vton/.env
pip install -r services/vton/requirements.txt
VTON_BACKEND=mock python app.py

Mở http://localhost:8000

2. Inference thật (CatVTON local)

cd services/vton
pip install torch torchvision --index-url https://download.pytorch.org/whl/cpu
pip install -r requirements.txt
bash download-models.sh
uvicorn app.main:app --reload --port 8000

CPU: 5–15 phút/ảnh. GPU (CUDA): nhanh hơn nhiều.

3. Replicate (cloud, chất lượng cao)

# Trong services/vton/.env
REPLICATE_API_TOKEN=r8_...
VTON_BACKEND=auto

4. Luồng sử dụng

  1. Trang chủ → chọn trang phục → Thử đồ
  2. Upload ảnh toàn thân (đứng thẳng, nền đơn giản)
  3. Nhấn Bắt đầu thử đồ → chờ job hoàn thành
  4. Xem kết quả và lịch sử tại /admin

5. Docker

docker compose up --build

Mặc định chạy VTON_BACKEND=mock. Mount volume virtulook_data cho storage.

6. Test

pip install -r requirements-dev.txt
python -m pytest
